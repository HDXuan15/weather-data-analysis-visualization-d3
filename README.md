# Weather D3 Project nhóm 3

Dự án trực quan hóa dữ liệu thời tiết bằng D3.js.

## Cách chạy

Mở terminal tại thư mục chứa project, sau đó chạy:

```bash
cd weather-data-analysis-visualization-d3 (nếu chưa vào đúng thư mục)
# chạy lệnh 
python -m http.server 8000 
# truy cập web để thấy giao diện
http://localhost:8000/
